----------
Install:
----------

Right click Windows' Start Button
  Select Windows PowerShell (Admin)

In Windows PowerShell ...
cd path_to_QoiShellExtensions
regsvr32 QoiShellExtensions.dll
exit

----------
Uninstall:
----------

Right click Windows' Start Button
  Select Windows PowerShell (Admin)

In Windows PowerShell ...
cd path_to_QoiShellExtensions
regsvr32 /u QoiShellExtensions.dll
exit
